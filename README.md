### Basics

ReactJS with Material-UI

```sh
git clone http://github.com/joltsama/dmr-frontend
cd dmr-frontend
yarn install
yarn start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

API Server runs on port 7000
